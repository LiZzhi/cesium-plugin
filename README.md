# cesium-plugin
感兴趣的可以 Star 支持一下,谢谢
## 简介
#### 目前已完成:
* plotTool 几何编辑
* pathRoaming 路径漫游
* mapvLayer mapv可视化
* linkView 二三维视图联动
* videoShed 视频投影
* layerSplit 卷帘分析
* gridRender 网格渲染
* effectPoint 特效点(更新中)
* cameraLimit 视角限制
* 持续更新中...
#### 特殊:
* emptyWork 为空工程,减少配置麻烦

## 使用方法
* 请在根目录安装 npm i
* 插件及Demo在子文件夹内，使用方法请查阅各子文件夹内README.md

* node 16.18.0
* npm 9.1.1
